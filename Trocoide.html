Trocoides

Se trata de las curvas trazados por un punto en un radio de una circunferencia, a una distancia d del centro, cuando esta rueda sin deslizar sobre una recta. En la figura, el radio es igual a 1. Si d = r, la curva trazada es una cicloide. Si d < 1, se trata de una cicloide acortada, y si d > 1, alargada. Si d = 0, se trata de una línea recta.

Para comparar varias, se puede activar el rastro hasta que se completen los tres ciclos, desactivarlo, modificar d, y volverlo activar. Todo ello, cuantas veces se quiera.

Ignacio Larrosa Cañestro (Grupo XeoDin), 3 marzo 2014. Creado con GeoGebra
