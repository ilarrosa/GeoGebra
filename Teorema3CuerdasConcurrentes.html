Teorema de las 3 cuerdas concurrentes

Si de un mismo punto parten tres cuerdas a, c y b (en este orden), tales que ∠(a,c) = α y ∠(b,c) = β, se tiene que:

c=(a senβ + b senα)/sen(α+β)

Se emplea el Teorema del seno para calcular los segmentos verdes y luego el Teorema de Ptolomeo al cuadrilátero resultante.
