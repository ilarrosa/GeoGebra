En el △ABC se tiene que a = 7, b = 9 y c = 12. Se toman puntos D y E en el lado BC de manera que ∠BAD = ∠DAE = ∠EAC = 1/3 ∠BAC.

Determinar el valor de ∠EDA.

cos(α) = cos(3δ) = (12²+9²-7²)/(2·22·9) = 22/27

cos(β) = (12²+7²-9²)/(2·12·7) = 2/3, sin(β) = √5/3

cos(3δ) = 4cos³(δ) - 3cos(δ) = 22/27, d = cos(δ)

108d³ - 81d - 22 = 0, solución racional d = -2/3, (2d + 3)(36d² - 24d + 11) = 0

d = (2 ± √15)/6 ⇒ cos(δ) = (2 + √15)/6, sen(δ) = (2√3 - √5)/6

cos(ε) = cos(β + δ) = cos(β)cos(δ) - sen(β)sen(δ) = 1/2 ⇒ ε = 60º
