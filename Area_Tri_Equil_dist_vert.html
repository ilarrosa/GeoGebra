Área triángulo equilátero a partir de distancias a los vértices

Determinar el área de un triángulo equilátero △ABC en función de las distancias a, b y c de un punto interior D a sus vértices.

A partir de esta expresión para el área es encillo estableces la relación simetrica entre el lado del triángulo equilátero y las tres distancias:
3(a⁴+b⁴+c⁴+d⁴) = (a²+b²+c²+d²)²    (Ver DistTriEqui.pdf)
