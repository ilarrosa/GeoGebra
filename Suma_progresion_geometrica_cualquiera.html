Suma de una progresión geométrica

Suma de una progresión geométrica de razón r, r > 0.
Utiliza el deslizador "Suma de ..." para variar el número de términos.
Si r < 1, desplazándolo a la derecha del todo, se suman todos los términos de la progresión.
El gráfico no permite mostrarlo, pero se aplican las mismas fórmulas para la suma si r < 0.
Desplaza el punto A para variar el término inicial a1.

¿Qué ocurre cuando r = 1? Prueba también con r = 0.8, por ejemplo.
