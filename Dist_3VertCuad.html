Lado de un cuadrado a partir de 3 distancias a los vértices

Se puede determinar el lado de un cuadrado dadas las distancias de un punto P a tres vértices. Se obtienen dos valores d y d', para uno de ellos P es interior y para el otro exterior al cuadrado.

Si se dan el lado del cuadrado y dos de las distancias, se puedn hallar las otras.

Si la distancias conocidas son a vértices opuestos, hay dos soluciones, a y a', correspondientes a vértices opuestos.

Si corresponden a dos vértices consecutivos, la distancia obtenida c es la opuesta a la tomada como b.

El applet, dados a, b y c, calcula analíticamente d y d'. Para a' basta tener en cuenta que:

a2 + a'2 = b2 + c2 (Distancias a los vértices de un rectángulo)
