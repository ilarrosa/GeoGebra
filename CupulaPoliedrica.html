Cúpulas poliedricas

Las cúpulas poliedricas triangular (J3), cuadrangular (J4) y pentagonal (J5) son prismatoides formados por una base de 2n lados, n cuadrados, n triángulos y otra base de n lados (n = 3, 4, 5), todos regulares y de la misma arista, como todos los solidos de Johnson.

Su volumen se calcula fácilmente descomponiéndolo en un prisma central de n caras laterales, n prismas triángulares y n pirámides triángulares.

También se puede usar la fórmula del volumen del prismatoide.
