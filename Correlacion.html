Correlación cuadrantes 1º - 3º

Se crean 1000 puntos al azar con x e y en [0, 1], o x e y en [-1, 0], y se calcula el coeficiente de correlación de Pearson

Este es un Applet de Java creado con GeoGebra desde www.geogebra.org – Java no parece estar instalado Java en el equipo. Se aconseja dirigirse a www.java.com

El resultado teórico es 0.75.
